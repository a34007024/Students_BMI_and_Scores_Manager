# Students_BMI_and_Scores_Manager
+ Students_BMI_and_Scores_Manager is a project for final teamwork of NTUNHS-Programming Design-108上
 
## 預期達成功能
+ 1.可輸入學生身高、體重、姓名、id、國英數成績的介面
+ 2.自動計算出BMI
+ 3.輸入數值防呆
+ 4.id自動編序
+ 5.資料庫連接
+ 6.圖形化介面的資料編輯環境
+ 7.自動隨機產生指定筆數的資料(自動隨機生成大數據)

## 若能力可及可能增加的功能
+ 1.國英數成績繪製成圖表
+ 2.分析上下學期成績及身高體重變化
+ 3.依照分析結果給出建議(小評語)
+ 4.匯出資料報表
+ 5.輸入特定資料跳出小遊戲(小彩蛋)
